02. CSS Architecture

CSS sadece stil vermek değildir. Box-Model, Flexbox ve Grid sistemleri ile matematiksel bir düzen oluşturmaktır. GPU hızlandırmalı animasyonlar için `will-change` ve `transform` kullanımı kritiktir.

transform: translate3d(0,0,0);

03. JavaScript Runtime

JS, V8 motoru üzerinde çalışır. Event Loop mekanizması sayesinde tek thread üzerinden asenkron işlemleri yönetir. Memory Leak (bellek sızıntısı) yönetimi ve verimli DOM manipülasyonu performansı belirler.
